Project Overview

This project focused on analyzing synovial fluid flow in hip joint replacements using advanced Computational Fluid Dynamics (CFD) tools such as Fusion 360 and Ansys Fluent. The primary goal was to assess design parameters and propose optimized solutions for improving joint durability and wear resistance.

Tools Used

  • Fusion 360
  • Ansys Fluent

Key Outcomes

The results provided insights into enhancing joint performance, contributing to improved prosthetic technology in healthcare.
